# Responsive Landing Page

This project is created as part of SkillCraft Technology Web Development Internship Task 1.

## Task Objective

Create an interactive navigation menu that:

- Changes color or style when scrolled
- Changes style when hovering over menu items
- Has a fixed position
- Is visible on all pages
- Includes responsive design

## Features

- Fixed Navigation Bar
- Scroll-based Navbar Color Change
- Hover Effects on Menu Items
- Responsive Landing Page
- Clean UI Design

## Technologies Used

- HTML
- CSS
- JavaScript
